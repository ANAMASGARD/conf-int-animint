# Confidence Interval Visualization

This repository contains a small interactive demo showing how frequentist confidence intervals behave across repeated samples. 
GSOC 2026 Medium Test Task. 

- **Live Link**: [https://anamasgard.github.io/conf-int-animint/](https://anamasgard.github.io/conf-int-animint/)
- **Source Code**: [https://github.com/ANAMASGARD/conf-int-animint/blob/main/conf_int_viz.R](https://github.com/ANAMASGARD/conf-int-animint/blob/main/conf_int_viz.R)

### What this app shows

- **90% confidence intervals** for a sequence of simulated samples.
